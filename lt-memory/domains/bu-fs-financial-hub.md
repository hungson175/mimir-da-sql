# BU FS: Financial Hub

**Domain ID:** `c80e0c61-0d52-4fd6-a2e5-9d2892da7955`

## Description
The dataset records information about Financial Hub miniapp, including:
(1) traffic login TTTC (Trung tâm tài chính) and QLTK (Quản lý tài khoản)
(2) Entrypoint user login FinHub (TTTC & QLTK) and from FinHub navigate to other miniapps
(3) List user retain monthly and weekly (retain FinHub overall and retain each miniapp TTTC and QLTK individually)
(4) Action click/ engage: Number user have action (impress/ engage) with FinHub by component
