## Part 3: Strategic Recommendation

### 3.1 The Opportunity

MoMo can build **"Túi Thần Tài AI"** — an AI-powered wealth companion that:

1. **Coaches users** to save more (like Stash Money Coach)
2. **Rounds up** transactions for micro-investing (like Acorns)
3. **Answers questions** about investing (like Magnifi)
4. **Recommends** appropriate financial products

### 3.2 Recommended Features (MVP)

| Feature | Description | Comparable To |
|---------|-------------|---------------|
| **AI Savings Coach** | Personalized tips to increase savings | Stash Money Coach |
| **Round-up Investing** | Auto-invest spare change from transactions | Acorns Round-Ups® |
| **Wealth Q&A** | Natural language answers about finance | Magnifi Assistant |
| **Goal-based Planning** | Save for specific goals (home, education, retirement) | Stash Goals |
| **Smart Portfolio** | AI-suggested ETF/stock allocations | Betterment |

### 3.3 Business Model Options

| Model | Pricing | Pros | Cons |
|-------|---------|------|------|
| **Freemium** | Basic free, premium AI features $2-5/month | High adoption | Lower ARPU |
| **Túi+ Integration** | Bundle with existing Túi+ subscription | Leverages existing base | Less focused |
| **AUM-based** | 0.5-1% of invested assets | Scales with user growth | Complex to implement |
| **Hybrid** | Free AI coach, fee for execution | Best of both | Requires careful positioning |

**Recommendation:** Start with freemium + Túi+ integration, then add AUM-based fees for managed portfolios.

### 3.4 Go-to-Market Phases

**Phase 1 (Months 1-3): Foundation**
- Launch AI Savings Coach in Moni
- Integrate with TTT balance data
- A/B test coaching messages

**Phase 2 (Months 4-6): Round-ups**
- Build round-up feature for transactions
- Auto-transfer to TTT or new investment account
- Partner with fund companies for ETF access

**Phase 3 (Months 7-12): Full Wealth Suite**
- Launch goal-based investing
- Add stock/ETF recommendations
- Build portfolio tracking across TTT + CVS
